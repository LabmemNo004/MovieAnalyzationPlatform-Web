# MovieAnalyzationPlatform-Web



# 开发规范

1. 统一在**develop**分支下进行提交

2. 每次提交前(最好在写代码前),先`pull`最新的版本,避免代码冲突

3. 每次提交说明修改了哪些地方

4. 遵循统一的代码风格

5. 提交时不需要pull request 直接提交就好，如果有冲突和那个写那个部分的同学协商
